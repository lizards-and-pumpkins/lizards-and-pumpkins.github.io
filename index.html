Catalog Without Trade-offs

Merchants have to make a hard decision between performance and consistency. The online shop catalog can be either fast or contain up to date stocks, prices and other product data.

Lizards & Pumpkins resolves this trade-off: It makes all product changes instant and delivers pages at a blazing speed.

Codebase Developers Won't Hate

Every line of code is written with a developer in mind. Every line of code is open-sourced. Every line of code is test driven. Flexible technology stack, no dependencies on 3rd party frameworks, readable code and a developer VM will make Lizards & Pumpkins fun to work with.

Pure Partnership

Lizards & Pumpkins is a platform. This means setup and support must be executed by a team of professional developers. We carefully select web agencies as our partners, who we recommend for incoming merchant inquiries.
